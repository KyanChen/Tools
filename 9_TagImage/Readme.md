# 本软件用于目标检测标注
# 采用TagImage.py进行目标标注
## 注意事项：
1. 标注前：
    1. 更改*inPathName*为图像所在路径
    2. 更改*classToWrite*为所标注目标类别
    3. 更改*widthRatio、heightRatio*为所需的宽高缩放因子
2. 具体操作：
    1. 使用鼠标进行标注
    2. d或D键调出下一标注图，并确认当前标注
    3. a或A键调出上一标注图，在当前图操作后按d或D键确认
    4. ESC键取消当前图像目标框的标注
